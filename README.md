# Barycentric Coordinates

A demonstration of the barycentric coordinates.

![Preview Video](./resources/preview.mp4)

## Controls
| Control | Description |
|:---:|:---:|
| Left Mouse | Move Camera / Vertices / Point |
| I | Toggle Inside Triangle |
| V | Toggle Vertices |
| A | Toggle Axes |
| R | Reset (All) |

## Build from Source
1. `npm install`
2. `npm run build`
